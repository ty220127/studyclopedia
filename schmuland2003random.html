無限列 $(\varepsilon_j)_{j=1}^{\infty}$ は，各項がそれぞれ独立に以下の分布に従う確率変数列であるとする：

\begin{align} P(\varepsilon_j = 1) = P(\varepsilon_j = -1) = \fraction{1}{2}. \end{align}

これを用いて，「符号のランダムな調和級数（以下，ランダム調和級数）」$X$ を以下の通り定義する：

\begin{align} X = \sum_{j=1}^{\infty} \fraction{\varepsilon_j}{j}. \end{align}


 

 

任意の $j = 1,\, 2,\, \cdots$ に対し，確率変数 $\varepsilon_j$ の期待値と２次モーメントはそれぞれ

\begin{align} E(\varepsilon_j) = 0,\quad E(\varepsilon_j^2) = 1 \end{align}

である．

証明

期待値の定義から直ちに計算できる：

\begin{align} E(\varepsilon_j) &= 1\cdot \fraction{1}{2} + (-1) \cdot \fraction{1}{2} = 0,\Rule{0px}{0px}{20px}\\ E(\varepsilon_j^2) &= 1^2\cdot \fraction{1}{2} +(-1)^2 \cdot \fraction{1}{2} = 1. \end{align}


 

 

以下で与えられる確率変数列 $S_n$ は，ほとんど確実に収束する．：

\begin{align} S_n = \sum_{j=1}^{n} \fraction{\varepsilon_j}{j}, \end{align}

つまり，$X = \limit_{n\to \infty} S_n$ はほとんど確実に存在する．

証明


 

 

$X$ の期待値と2次モーメントは

\begin{align} E(X) = 0,\quad E(X^2) = \fraction{\pi^2}{6} \end{align}

である．

証明

$E(X)$ は，期待値の線形性と の結果から直ちに得られる：

\begin{align} E(X) &= E\left(\sum_{j=1}^{\infty}\fraction{\varepsilon_j}{j}\right) = \sum_{j=1}^{\infty} \fraction{E(\varepsilon_j)}{j} = 0. \end{align}

$E(X^2)$ は上記に加え，$(\varepsilon_j)_{j=1}^{\infty}$ の独立性を利用することで計算される：

\begin{align} E(X^2) &= E\left( \sum_{j=1}^{\infty}\fraction{\varepsilon_j}{j} \sum_{k=1}^{\infty} \fraction{\varepsilon_k}{k}\right) = E\left( \sum_{j=1}^{\infty}\sum_{k=1}^{\infty} \fraction{\varepsilon_j \varepsilon_k}{jk}\right)\\ &= \sum_{j=1}^{\infty}\sum_{k=1}^{\infty} \fraction{E(\varepsilon_j \varepsilon_k)}{jk} = \sum_{j=1}^{\infty} \left(\fraction{E(\varepsilon_j^2)}{j^2} + \sum_{\substack{k=1\\ k\not = j}}^{\infty}\fraction{E(\varepsilon_j \varepsilon_k)}{jk}\right)\\ &= \sum_{j=1}^{\infty} \fraction{1}{j^2} + \underset{\hspace{30mm}=0}{\underline{\sum_{j=1}^{\infty}\sum_{\substack{k=1\\ k\not =j}}^{\infty} \fraction{E(\varepsilon_j)E(\varepsilon_k)}{jk}}} = \sum_{j=1}^{\infty}\fraction{1}{j^2} = \fraction{\pi^2}{6}. \end{align}

補足

ここで挙げられた命題より，直ちに $X$ の分散 $V(X)$ を計算できる：

\begin{align} V(X) = E(X^2) - E(X)^2 = E(X^2) = \fraction{\pi^2}{6}. \end{align}
