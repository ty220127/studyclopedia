無限列 $(\varepsilon_j)_{j=1}^{\infty}$ は，各項がそれぞれ独立に以下の分布に従う確率変数列であるとする：

\begin{align} P(\varepsilon_j = 1) = P(\varepsilon_j = -1) = \fraction{1}{2}. \end{align}

これを用いて，「符号のランダムな調和級数（以下，ランダム調和級数）」$X$ を以下の通り定義する：

\begin{align} X = \sum_{j=1}^{\infty} \fraction{\varepsilon_j}{j}. \end{align}


 

 

任意の $j = 1,\, 2,\, \cdots$ に対し，確率変数 $\varepsilon_j$ の期待値と２次モーメントはそれぞれ

\begin{align} E(\varepsilon_j) = 0,\quad E(\varepsilon_j^2) = 1 \end{align}

である．

証明

期待値の定義から直ちに計算できる：

\begin{align} E(\varepsilon_j) &= 1\cdot \fraction{1}{2} + (-1) \cdot \fraction{1}{2} = 0,\Rule{0px}{0px}{20px}\\ E(\varepsilon_j^2) &= 1^2\cdot \fraction{1}{2} +(-1)^2 \cdot \fraction{1}{2} = 1. \end{align}


 

 

以下で与えられる確率変数列 $S_n$ は，ほとんど確実に収束する．：

\begin{align} S_n = \sum_{j=1}^{n} \fraction{\varepsilon_j}{j}, \end{align}

つまり，$X = \limit_{n\to \infty} S_n$ はほとんど確実に存在する．

証明


 

 

$X$ の期待値と2次モーメントは

\begin{align} E(X) = 0,\quad E(X^2) = \fraction{\pi^2}{6} \end{align}

である．

証明

$E(X)$ は，期待値の線形性と の結果から直ちに得られる：

\begin{align} E(X) &= E\left(\sum_{j=1}^{\infty}\fraction{\varepsilon_j}{j}\right) = \sum_{j=1}^{\infty} \fraction{E(\varepsilon_j)}{j} = 0. \end{align}

$E(X^2)$ は上記に加え，$(\varepsilon_j)_{j=1}^{\infty}$ の独立性を利用することで計算される：

\begin{align} E(X^2) &= E\left( \sum_{j=1}^{\infty}\fraction{\varepsilon_j}{j} \sum_{k=1}^{\infty} \fraction{\varepsilon_k}{k}\right) = E\left( \sum_{j=1}^{\infty}\sum_{k=1}^{\infty} \fraction{\varepsilon_j \varepsilon_k}{jk}\right)\\ &= \sum_{j=1}^{\infty}\sum_{k=1}^{\infty} \fraction{E(\varepsilon_j \varepsilon_k)}{jk} = \sum_{j=1}^{\infty} \left(\fraction{E(\varepsilon_j^2)}{j^2} + \sum_{\substack{k=1\\ k\not = j}}^{\infty}\fraction{E(\varepsilon_j \varepsilon_k)}{jk}\right)\\ &= \sum_{j=1}^{\infty} \fraction{1}{j^2} + \underset{\hspace{30mm}=0}{\underline{\sum_{j=1}^{\infty}\sum_{\substack{k=1\\ k\not =j}}^{\infty} \fraction{E(\varepsilon_j)E(\varepsilon_k)}{jk}}} = \sum_{j=1}^{\infty}\fraction{1}{j^2} = \fraction{\pi^2}{6}. \end{align}

補足

ここで挙げられた命題より，直ちに $X$ の分散 $V(X)$ を計算できる：

\begin{align} V(X) = E(X^2) - E(X)^2 = E(X^2) = \fraction{\pi^2}{6}. \end{align}


 

 

$X$ の $t$ に関する積率母関数 $E(\exp(tX))$ に対して

\begin{align} E(\exp(tX)) \leq \exp\left( \fraction{t^2\pi^2}{12} \right) \end{align}

が成り立つ．

証明

$E(\exp(tX))$ は以下のように書き換えられる：

\begin{align} &E(\exp(tX)) = E\left( \exp\left( t \sum_{j=1}^{\infty}\fraction{\varepsilon_j}{j} \right) \right)\\ &= E\left(\prod_{j=1}^{\infty} \exp\left(\fraction{t\varepsilon_j}{j}\right) \right)\\ &= \prod_{j=1}^{\infty} E\left(\exp\left(\fraction{t\varepsilon_j}{j}\right)\right)\\ &= \prod_{j=1}^{\infty} \left( P(\varepsilon_j = +1) \exp\left(\fraction{t\cdot 1}{j} \right) +P(\varepsilon_j = -1) \exp\left( \fraction{t\cdot (-1)}{j}\right) \right)\\ &= \prod_{j=1}^{\infty} \fraction{1}{2}\left( \exp\left(\fraction{t}{j} \right) + \exp\left( -\fraction{t}{j}\right) \right). \end{align}

ここで，指数関数の Maclaurin 展開を考えると，任意の実数 $x$ に対して

\begin{align} \exp(x) = \sum_{k=0}^{\infty} \fraction{1}{k!} x^k \end{align}

である．よって $x$ に $-x,\ x^2/2$ を代入することでそれぞれ

\begin{align} \exp(-x) = \sum_{k=0}^{\infty} \fraction{(-1)^k}{k!} x^k,\quad \exp\left(\fraction{x^2}{2}\right) = \sum_{k=0}^{\infty} \fraction{1}{2^k k!}x^{2k} \end{align}

を得る．よって

\begin{align} &\exp\left(\fraction{x^2}{2}\right) - \fraction{1}{2}(\exp(x) + \exp(-x))\\ &= \sum_{k=0}^{\infty} \fraction{1}{2^k k!}x^{2k} - \fraction{1}{2} \sum_{k=0}^{\infty}\fraction{1 +(-1)^k}{k!} x^k\\ &= \sum_{k=0}^{\infty} \fraction{1}{2^k k!}x^{2k} - \sum_{m=0}^{\infty} \fraction{1}{(2m)!} x^{2m}\\ &= \sum_{m=0}^{\infty} \underset{m=0\, \Rightarrow\, 0}{\underline{\left(\fraction{1}{2^m m!} - \fraction{1}{(2m)!} \right) }}x^{2k}\\ &= \sum_{m=1}^{\infty} \fraction{1}{k!} \left(\fraction{1}{2^k} -\prod_{\ell = m+1}^{2m} \fraction{1}{\ell} \right) x^{2k} \geq 0, \end{align}

つまり

\begin{align} \fraction{1}{2}(\exp(x) +\exp(-x)) \leq \exp\left(\fraction{x^2}{2}\right) \end{align}

が成り立つ．これを用いることで，$E(\exp(tX))$ は以下のように評価される：

\begin{align} E(\exp(tX)) &= \prod_{j=1}^{\infty} \fraction{1}{2} \left( \exp\left(\fraction{t}{j}\right) + \exp\left(-\fraction{t}{j}\right)\right)\\ &\leq \prod_{j=1}^{\infty} \exp\left(\fraction{t^2}{2j^2}\right) = \exp \left( \fraction{t^2}{2} \prod_{j=1}^{\infty}\fraction{1}{j^2}\right)\\ &= \exp \left( \fraction{t^2}{2}\cdot \fraction{\pi^2}{6} \right) = \exp \left( \fraction{t^2 \pi^2}{12} \right). \end{align}


 

 

$X$ がある正数 $x \gt 0$ より大きくなる確率について

\begin{align} P(X \gt x) \leq \exp\left( -\fraction{3x^2}{\pi^2} \right) \end{align}

が成り立つ．

証明

まず，指数関数の性質より，任意の $t\in \mathbb{R}$ に対して $X \gt x$ と $\exp(tX) \gt \exp(tx)$ は同値である．つまり，任意の $t\in \mathbb{R}$ に対して

\begin{align} P(X\gt x) = P(\exp(tX) \gt \exp(tx)) \end{align}

である．ここで Markov の不等式より

\begin{align} &P(\exp(tX) \gt \exp(tx)) \leq \fraction{E(tX)}{\exp(tx)}\\ &\underset{\text{[12-5]}}{\leq} \exp\left(\fraction{t^2\pi^2}{12} \right) \exp(-tx)\\ &= \exp\left( \fraction{t^2\pi^2}{12} -tx \right) = \exp \left( \fraction{\pi^2}{12} \left(t -\fraction{6x}{\pi^2}\right)^2 -\fraction{3x^2}{\pi^2}\right) \end{align}

が成り立つ．よって，任意の $t\in \mathbb{R}$ に対して

\begin{align} P(X\gt x) \leq \exp \left( \fraction{\pi^2}{12} \left(t^2 -\fraction{6x}{\pi^2}\right)^2 -\fraction{3x^2}{\pi^2}\right) \end{align}

であり，この不等式における指数関数の引数の最小値は $t= 6x/\pi^2$ のときに $-3x^2/\pi^2$ であるから

\begin{align} P(X\gt x) \leq \exp\left(-\fraction{3x^2}{\pi^2} \right) \end{align}

が成立する．
