ものぐさランダムウォークとは，ウォーカーがその場に留まる確率を持つモデルであるが，古典アルゴリズムにおいて有用な道具である．本論文では，その量子版である「無気力量子ウォーク」を各頂点で $l$ 個の自己閉路を持つ状況下で提案し，$N$ 頂点完全グラフ上のマーク頂点の探索を定式化した Grover のアルゴリズムへの効果を調査する．位相反転コインを用いた離散時間量子ウォークにおいては，各頂点に１つずつ自己閉路を追加することにより，探索の成功確率を $1/2$ から $1$ に上げることができる．しかし，自己閉路を更に追加することは，成功確率の減少に繋がる．代わりに Shenvi, Kempe, and Whaley (2003) のコインを用いると，自己閉路の追加は単に探索を遅くしてしまう．これらのコインは，前者は $l$ が $N$ 以下のときに古典アルゴリズムより速いが，後者は $l$ が $N^2$ 以下である必要がある点で異なる．最後に，連続時間量子ウォークはこれらの離散時間の例の何れとも異なっており，自己閉路を与えても違いが発生しない．これらのふるまいは，複数頂点の場合にも一般化できる．


 

 

頂点に対応する Hilbert 空間を位置 $1,\,2,\,\cdots,\,N$ に対応するベクトル群による計算基底 $\{\ket{1},\,\ket{2},\,\cdots,\,\ket{N}\}$ の張る線形空間 $\mathcal{H}_v$，ウォーカーの推移に関する「コイン」に対応する Hilbert 空間を内部状態 $1,\,2,\,\cdots,\, N-1$ に対応するベクトル群による計算基底 $\{\ket{1},\,\ket{2},\,\cdots,\,\ket{N-1}\}$ の張る線形空間 $\mathcal{H}_c$ とする：

\begin{align} \mathcal{H}_v &:= \spn\{\ket{i}\,|\,i = 1,\,2,\,\cdots,\,N\} \simeq \mathbb{C}^N,\\ \mathcal{H}_c &:= \spn\{\ket{e}\,|\,e = 1,\,2,\,\cdots,\,N-1\} \simeq \mathbb{C}^{N-1}. \end{align}

このとき，それぞれの空間に対応する一様状態は

\begin{align} \ket{s_v} = \fraction{1}{\sqrt{N}}\sum_{i=1}^{N}\ket{i},\quad \ket{s_c} = \fraction{1}{\sqrt{N-1}} \sum_{e=1}^{N-1}\ket{e} \end{align}

となり，これらを用いて初期状態を

\begin{align} \ket{\psi_0} = \ket{s_v}\otimes \ket{s_c} \end{align}

と定める．量子ウォーク探索の作用素 $U$ は

\begin{align} U = S\cdot \{(I_n -\ketbra{w}{w})\otimes C_0 + \ketbra{w}{w}\otimes C_1\} \end{align}

と表される．ここで $\ket{w}\in \mathbb{C}^N$ はマーク頂点 $w\in [N]:=\{1,\,2,\,\cdots,\,N\}$ により添え字づけられたベクトルである．また $S$ はフリップフロップシフト作用素

\begin{align} S = \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N} \ketbra{j}{i}\otimes \ketbra{(i-j)\modulo N}{(j-i)\modulo N}, \end{align}

$C_0$ は Grover 拡散コイン

\begin{align} C_0 = 2\ketbra{s_c}{s_c} -I_{N-1} \end{align}

であり，$C_1$ はマーク頂点に対してコインとして導入されるあるユニタリ行列とする．なお，本サイトでは $w=1$ とする（このように設定しても一般性を失わない）．

補足１

論文では Hilbert 空間を $\mathbb{C}^N,\ \mathbb{C}^{N-1}$ としているが，必ずしもそのように設定する必要はなく，また $\mathbb{C}^{N},\ \mathbb{C}^{N-1}$ とおくことに依る処理もないので，本サイトでは一般の Hilbert 空間として設定した．

補足２

論文ではある頂点 $i$ から $j$ への推移に対応するベクトルを $\ket{i\to j}$ とおいているが，これが具体的に $\mathcal{H}_c$ のどのようなベクトルなのかが述べられていない．この設定には様々な方法があるが，本サイトでは処理のしやすさを考慮し $\ket{i\to j} = \ket{(j-i)\modulo N}$ と設定した．この well-definedness を以下で確認する．

$S$ の well-definedness

$S$ が Hermite 作用素でありかつユニタリ作用素であることを確認する．まず，以下の議論により $S$ は Hermite 作用素である：

\begin{align} S^\dagger &= \Biggl(\sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N} \ketbra{j}{i}\otimes \ketbra{(i-j)\modulo N}{(j-i)\modulo N}\Biggr)^\dagger\\ &= \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N} (\ketbra{j}{i}\otimes \ketbra{(i-j)\modulo N}{(j-i)\modulo N})^\dagger\\ &= \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N} \ketbra{i}{j}\otimes \ketbra{(j-i)\modulo N}{(i-j)\modulo N}\\ &= \sum_{j = 1}^{N}\sum_{\substack{i=1\\ i\not = j}}^{N} \ketbra{i}{j}\otimes \ketbra{(j-i)\modulo N}{(i-j)\modulo N}\\ &= S. \end{align}

また，

\begin{align} &S^\dagger S = SS^\dagger = S^2 \\ &= \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N}\sum_{i' = 1}^{N}\sum_{\substack{j'=1\\ j'\not = i'}}^{N} \ket{j}\braket{i|j'}\bra{i'}\\ & \hspace{5mm}\otimes \ket{(i-j)\modulo N}\braket{(j-i)\modulo N|(i'-j')\modulo N}\bra{(j'-i')\modulo N}\\ &= \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N}\sum_{i' = 1}^{N}\sum_{\substack{j'=1\\ j'\not = i'}}^{N} \delta_{ij'} \delta_{((j-i)\modulo N)((i'-j')\modulo N)} \ketbra{j}{i'}\\ & \hspace{50mm} \otimes \ketbra{(i-j)\modulo N}{(j'-i')\modulo N}\\ &= \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N}\sum_{\substack{i' = 1\\ i'\not =i}}^{N} \delta_{ji'} \ketbra{j}{i'} \otimes \ketbra{(i-j)\modulo N}{(i -i')\modulo N}\\ &= \sum_{i = 1}^{N}\sum_{\substack{j=1\\ j\not = i}}^{N}\ketbra{j}{j} \otimes \ketbra{(i-j)\modulo N}{(i -j)\modulo N}\\ &= \sum_{j = 1}^{N}\ketbra{j}{j} \otimes \sum_{\substack{i=1\\ i\not = j}}^{N}\ketbra{(i-j)\modulo N}{(i -j)\modulo N}. \end{align}

ここで，任意の $j = 1,\,\cdots,\,N$ に対し，以下が成り立つ：

\begin{align} &\sum_{\substack{i=1\\ i\not =j}}{N} \ketbra{(i-j)\modulo N}{(i-j)\modulo N}\\ &= \sum_{i=1}^{j-1}\ketbra{N+i-j}{(N+i-j} + \sum_{i=j+1}^{N}\ketbra{i-j}{i-j}\\ &= \underset{e := N+i-j}{\underline{\sum_{e=N-j+1}^{N-1} \ketbra{e}{e}}} + \underset{e := i-j}{\underline{\sum_{e = 1}^{N-j} \ketbra{e}{e}}} = \sum_{e=1}^{N-1} \ketbra{e}{e}. \end{align}

よって，

\begin{align} S^2 &= \sum_{j = 1}^{N}\ketbra{j}{j} \otimes \sum_{\substack{i=1\\ i\not = j}}^{N}\ketbra{(i-j)\modulo N}{(i -j)\modulo N}\\ &= \sum_{j=1}^{N}\ketbra{j}{j} \otimes \sum_{e=1}^{N-1}\ketbra{e}{e}\\ &= I_N\otimes I_{N-1} = I_{N(N-1)}. \end{align}

よって，$S$ はユニタリ作用素である．
